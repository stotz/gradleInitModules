# gradleInit Modules

Official modules for gradleInit.

## Structure

```
resolvers/       - Version resolvers (Maven Central, etc.)
integrations/    - Framework integrations (future)
```

## Security

This repository is signed. Verify with:

```bash
gradleInit verify --repo . --key official
```

## Available Modules

### Resolvers

- **maven_central** - Query Maven Central for latest versions
